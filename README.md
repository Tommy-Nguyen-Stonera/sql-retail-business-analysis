# SQL Retail Business Analysis

[View Interactive Report](https://htmlpreview.github.io/?https://raw.githubusercontent.com/Tommy-Nguyen-Stonera/sql-retail-business-analysis/main/SQL%20Retail%20Business%20Analysis%20Report.html)

Multi-year (2014-2016) sales analysis for a US retail business using T-SQL. Covers revenue trends, discount impact, customer segments, employee performance, and dynamic pivot queries across 9,994 orders.

## Key Findings

- Discounts over 50% produce losses across all product categories
- Technology leads profit for Consumer and Corporate segments
- Highest-revenue category is not always the most profitable within each segment
- Most employees get 60-80% of profit from a single product category
- State-level profit is concentrated in a small number of high-population states

SQL Server, T-SQL, Excel, CSV
